# Product Images Setup

Please add the following product images to the root directory of your project:

## Required Images:

1. **ultimate-greens.jpg**
   - Product: Ultimate Greens + Antioxidants
   - Recommended size: 800x800px or higher
   - Format: JPG or PNG

2. **ultimate-cleanse.jpg**
   - Product: Ultimate Cleanse
   - Recommended size: 800x800px or higher
   - Format: JPG or PNG

3. **zonefuel.jpg**
   - Product: ZoneFuel Prime Hydration
   - Recommended size: 800x800px or higher
   - Format: JPG or PNG

4. **collagen-peptides.jpg**
   - Product: Collagen Peptides Type I & III
   - Recommended size: 800x800px or higher
   - Format: JPG or PNG

## Instructions:

1. Save your product images from the screenshots/designs you have
2. Rename them to match the exact filenames above
3. Place them in the same directory as `index.html`
4. The website will automatically display them

## Image Optimization Tips:

- Use high-quality images (at least 800x800px)
- Optimize file size for web (aim for under 200KB per image)
- Use consistent aspect ratios for all products
- Consider using WebP format for better compression

Once images are added, your products will display beautifully on the homepage!
